# Expense Tracker CLI

Expense Tracker CLI adalah aplikasi **Command Line Interface (CLI)** untuk mencatat dan mengelola pengeluaran harian.  
Aplikasi ini berjalan langsung dari terminal dan menyimpan data secara lokal.

Project ini cocok untuk latihan backend / CLI development dan dapat dikembangkan lebih lanjut.
